How to Use Examples

Testing Procedure

  1. Download Images: Click the download buttons above to save example images to your device
  2. Return to Analysis: Go back to the main ErosionWatch interface
  3. Upload Examples: Use the drag-and-drop upload area to select the downloaded images
  4. Review Results: Compare the automated analysis results with the expected values listed above
  5. Interpret Recommendations: Examine the generated risk maps and planting suggestions
  6. Export Reports: Test the PDF and CSV export functionality using the analysis results

Note: These synthetic images are designed to demonstrate system capabilities. In actual field use, photo quality, lighting conditions, and reference object positioning will affect measurement accuracy. Always follow the analysis guidelines for optimal results.
